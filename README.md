# AI Interview Assistant & Candidate Evaluation System

An advanced Natural Language Processing (NLP) application designed to automate technical screening pipelines. The system parses resumes, extracts technical skills using Named Entity Recognition (NER), conducts a dynamic multi-stage technical interview using LLMs, and algorithmically grades candidate responses via semantic similarity.
